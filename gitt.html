Tutorial on How to Make a Github Page

1. Membuat repository baru pada akun github, dengan nama username.github.io. Disini repository disetting public dan tanpa README.md.

2. Setelah memastikan semua file siap, lakukan push folder ke dalam folder repositry baru tadi.

3. Jika file sudah tersedia di repository github, jangan lupa untuk mengubah source menjadi main/master.

5. Github page siap diakses melalui link yang tersedia (username.github.io). Untuk diingat, halaman web akan bisa diakses jika ada file index.html.
